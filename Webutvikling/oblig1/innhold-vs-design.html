Innhold vs Design

Hva er innhold

Innhold er er det du legger inn i HTML filen, det er det innholdet som brukeren vil se, uavhengig av hvordan siden er pyntet på med CSS

Hva er design?

Design er hvordan siden ser ut for sluttbrukeren. Design er vanligvis også kalt for pynting på siden. Dette er vanligvis laget i en CSS fil som sier hvor ting er posisjonert i forhold til hverander, samt farger, skrifttype og mye mer.

Hva er forskjellen på innhold og design

Forskjellen på innhold og design er at uansett hvordan du lager designet i en CSS fil, så vil brukeren uansett kunne lese innholdet som ligger i en HTML fil. Om nettleseren har problemer med å lese CSS filen og/eller CSS filen ikke er tilgjengelig, så vil leseren fortsatt se innholdet på siden.
