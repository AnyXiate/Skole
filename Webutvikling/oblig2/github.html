Git og Github

Hva er Git og Github

Git er en populær gratis distribuert versjonshåndteringssystem (VCS) som er brukt for å hente og administrere filer når flere samarbeider med små og store prosjekter innen kodeutvikling. Github derimot er en web-basert hosting service som samler Git repositories for at andre kan hente de.

Fordeler med å bruke Git

Fordeler med å bruke Git er blant annet at det fungerer raskt. Henting av filer og prosjekter går raskt fordi nesten alt er lokalt.

Alle som jobber med et prosjekt har en kopi av 'repoet'.

Git er også skalerbart, altså du kan ha så mange du vil til å jobbe med et prosjekt samtidig

Hva jeg syns om Github

Jeg er ny i bruken av github og har ikke så mye erfaring med det, men sålangt virker det ganske enkelt. Er et veldig enkelt verktøy til å bruke som backup for all koden jeg skriver. Vil absolutt annbefale andre å bruke det.
